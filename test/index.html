How It Works

This page demonstrates how any website can integrate Epistery by simply including the script tag:

<script type="module">
  import Witness from '/.epistery/lib/witness.js';
  const witness = await Witness.connect();
</script>

The Epistery routes (/.epistery/*) are automatically available when you include the Epistery library in your Express.js application.
